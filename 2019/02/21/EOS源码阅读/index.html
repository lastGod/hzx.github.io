EOS区块链相关数据结构

block header

block header的定义,来自于”chain\webassembly\block_header.hpp”文件中,其主要结构如下:
| 属性 | 大小(Byte) | 名称 |
| :——– | :—–: | :—-: |
| 时间戳 | 4 | 生产区块的时间 |
| 生产者 | 8 | 生产区块的超级节点 |
| 确认数 | 2 | 区块的确认数 |
| 交易的Merkle Root | 32 | 区块中全部交易的默克尔数的哈希值 |
| action的Merkel Root | 32 | 区块中全部action的默克尔数的哈希值 |
| schedule_version | 4 | 见证人排序版本号 |
| 下一个见证人 | - | 区块的下一个见证人,可以为空 |
| 扩展类型 | - | vector<std::pair<uint16_t,vector>> |
